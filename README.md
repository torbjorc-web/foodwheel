# FoodWheel Board Slides

## Overview
This project analyzes FoodWheel restaurant and order data to help answer questions about cuisine offerings, order trends over time, and customer spending.

## Objectives
- Show what cuisines FoodWheel offers.
- Analyze how average order amount changes over time.
- Measure how much customers have spent over the past six months.
- Visualize restaurant counts by neighborhood as an extra challenge.

## Files
- `restaurants.csv` — restaurant data.
- `orders.csv` — order data.
- `foodwheel_project.ipynb` — main notebook.

## Visualizations
- Pie chart of cuisine offerings.
- Bar chart of average order amount by month with error bars.
- Histogram of customer spending.
- Bar chart of restaurant count by neighborhood.

## Tools Used
- Python
- pandas
- matplotlib

## How to Run
1. Open `foodwheel_project.ipynb`.
2. Run all cells in order.
3. Make sure `restaurants.csv` and `orders.csv` are in the same folder.
